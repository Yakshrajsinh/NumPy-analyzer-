# **NumPy Array Analyzer**

A Python-based interactive CLI tool that allows users to create, manipulate, analyze, and perform mathematical operations on 1D, 2D, and 3D NumPy arrays.  
Designed as a learning + utility project for understanding NumPy operations hands-on.
