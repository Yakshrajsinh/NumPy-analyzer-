# NumPy Array Analyzer

This project is an interactive Python-based NumPy utility that allows users to create, manipulate, analyze, search, slice, filter, combine, split, and perform mathematical/statistical operations on 1D, 2D, and 3D NumPy arrays through a command-line interface.
